
# Mazia Version 2

Mazia is a modern AI Call Agent web application designed to manage, monitor, and visualize intelligent voice assistants with clarity and efficiency.

Version 2 focuses on a clean white & blue theme.


## Tech Stack

React

Vite

Tailwind CSS

### Installation

# Clone the repository

git clone https://github.com/Itsnotxarwa/mazia.git

## Navigate to the project folder

cd mazia

### Install dependencies

npm install

#### Run the development server

npm run dev


Open http://localhost:5173 in your browser to view the website.
